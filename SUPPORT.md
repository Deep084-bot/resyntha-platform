## Before Opening an Issue

1. Check the [troubleshooting guide](docs/troubleshooting.md)
2. Search existing [issues](https://github.com/anomalyco/resyntha/issues) and [discussions](https://github.com/anomalyco/resyntha/discussions)
3. Make sure you're on the latest version
4. Include relevant logs, error messages, and steps to reproduce
